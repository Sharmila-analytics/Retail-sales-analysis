# Retail Sales Analysis & Prediction

## Overview
This project analyses retail sales data to identify patterns, key factors affecting sales, and opportunities for business improvement.

## Tools & Technologies
- Python (Pandas, NumPy, Scikit-learn)
- Power BI
- Data Visualisation

## Key Analysis
- Analysed 500K+ transaction records
- Identified relationships between price, visibility, and sales
- Explored category-wise and store-wise performance

## Model Insights
- Applied regression and ensemble models to explore predictive patterns
- Gradient Boosting performed best among tested models

## Business Insights
- Higher-priced products tend to generate higher revenue
- Certain product categories outperform others
- Customer engagement and visibility significantly impact sales

## Conclusion
The analysis provides actionable insights for pricing strategies, marketing improvements, and customer retention.
