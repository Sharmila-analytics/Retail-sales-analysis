# Retail Sales Analysis & Prediction

## Overview
This project analyses retail sales data to identify key factors influencing sales performance and provide actionable business insights.

## Tools & Technologies
- Python (Pandas, NumPy, Scikit-learn)
- Data Visualisation (Matplotlib, Seaborn)
- Power BI (for dashboards)

## Key Analysis
- Analysed 500K+ retail transaction records  
- Explored relationships between price, visibility, and sales  
- Performed category-wise and store-wise analysis  

## Model Insights
- Applied Linear Regression, Random Forest, and Gradient Boosting  
- Gradient Boosting performed best among tested models  
- Models used to explore patterns rather than high-accuracy prediction  

## Key Business Insights
- High-priced products contribute significantly to revenue  
- Product visibility strongly impacts sales performance  
- Certain categories consistently outperform others  
- Some stores underperform and require targeted improvement  

## Business Impact
These insights can help businesses:
- Optimise pricing strategies  
- Improve product placement and visibility  
- Focus on high-performing categories  
- Increase overall revenue and customer engagement  

## Conclusion
This project demonstrates how data analysis and machine learning can support better business decision-making in retail environments.
